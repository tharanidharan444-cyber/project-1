This project uses Linear Regression to predict a student's final score based on:

- Hours Studied
- Attendance
- Assignments Completed
- Previous Score

The model is trained using historical student performance data and generates score predictions for new records.


tools
- Python
- Pandas
- Scikit-Learn



- Linear Regression


1. Load student dataset
2. Select input features
3. Split data into training and testing sets
4. Train Linear Regression model
5. Predict final scores
6. Evaluate model performance using MAE and R² Score



- Mean Absolute Error (MAE)
- R² Score


The model predicts student final scores and displays:

- MAE Value
- R² Score
